# Undo Close Tab for Chromium

Restore the most recently closed tab using **⌘Z** on macOS or **Ctrl+Z** on Windows.

Designed for Chromium browsers including Brave, Chrome, Edge, Vivaldi and Opera.

---

## Features

- Restore the most recently closed tab
- Works with Brave, Chrome, Edge, Vivaldi and Opera
- Does not interfere with Undo while typing in text fields
- Lightweight and fast
- No tracking
- No analytics
- No data collection

---

## Installation

1. Download or clone this repository.
2. Open:

```
brave://extensions
```

(or `chrome://extensions`)

3. Enable **Developer mode**.
4. Click **Load unpacked**.
5. Select the `src` folder.

---

## Keyboard Shortcut

Open your browser's:

**Extensions → Keyboard Shortcuts**

Assign your preferred shortcut.

Recommended:

- macOS: **⌘Z**
- Windows/Linux: **Ctrl+Z**

---

## Privacy

Undo Close Tab does **not**:

- collect data
- track users
- connect to external servers
- require an account

Everything runs locally inside your browser.
